************
Logging GELF
************

A python logging bundle to send logs using GELF. This is a rewrote of `Djehouty <https://github.com/ovh/djehouty>`_

----------
Quickstart
----------

First, install logging-gelf using `pip <https://pip.pypa.io/en/stable/>`_::

    pip install logging-gelf
